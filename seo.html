    ¿Qué es Seo?

    Es un conjunto de acciones orientadas a mejorar el posicionamiento de un sitio web en la lista de resultados de Google, Bing, u otros buscadores de internet.

    ¿Cómo hacer una web optimizada para SEO?

    • Registrar y verificar tu sitio web en Google Search Console, una herramienta gratuita de Google que permite a los propietarios de sitios web supervisar y gestionar su presencia en los resultados de búsqueda de Google.
    • Hacer un pequeño estudio de palabras clave
    • Crear una buena estructura de url's que contengan tu palabra clave
    • Tener los Títulos optimizados
    • Redactar descripciones con la palabra clave con texto atractivo para convertir
    • Optimizar las imágenes: nombre fichero, título alt, tamaño en pixels y su peso.
    • El contenido: original, natural y con las palabras clave en títulos y texto.
    • Enlaces: escribir palabras clave en el «anchor text»
    • Otros aspectos: Buen diseño web, promociona contenidos, crea lista suscriptores y haz networking, ves a eventos relacionados con tu negocio.

    Meta Etiquetas

    Las etiquetas meta en HTML sirven como piezas importantes de información para los motores de búsqueda. Se utilizan para describir elementos HTML y comunicar el contenido de un sitio web para los motores de búsqueda.

    En la siguiente tabla podemos observar algunos ejemplos y una pequeña descripción de las mismas:

    Ejemplo Función
    Meta Etiqueta de Descripción Esta meta etiqueta tiene el propósito de mostrar un resumen del contenido de nuestra pagina web. Su principal uso es al momento de mostrar tu página web en el motor de búsqueda, ya que este muestra dicha descripción debajo del título de la página, Debe ser atractiva y contener palabras claves que ayudarán a posicionar dicho sitio web
    Meta Etiqueta de Palabras Claves Esta meta etiqueta tiene el propósito de listar las palabras claves relevantes para el contenido de la página. En un principio era utilizado por los motores de búsqueda para lograr comprender sobre qué va a tratar la página
    Meta Etiqueta de Robots El propósito es indicar a los motores de búsqueda como indexar la página web. Los valores comunes que incluyen son: index, follow, noindex, entre otros
    Meta Etiqueta ViewPort Esta meta etiqueta tiene el propósito de controlar cómo se muestra la página en dispositivos móviles. Ayuda a que la página sea responsiva. Un valor común es content="width=device-width, initial-scale=1", este ajusta el ancho de la página al ancho del dispositivo y establece un zoom inicial adecuado
    Meta Etiqueta de Autor El propósito es brindar una información sobre el autor del contenido de la página. Es útil para así dar crédito al autor y puede ser utilizado por algunos motores de búsqueda y navegadores para mostrar esta información

    En conclusión, las meta etiquetas son una herramienta fundamental para optimizar la manera en que los motores de búsqueda interpretan y clasifican nuestro sitio web. Aunque no son visibles para los usuarios, su correcta utilización puede mejorar tanto el posicionamiento en los resultados de búsqueda como la experiencia del usuario en dispositivos móviles. Cada tipo de meta etiqueta tiene un propósito específico que contribuye al éxito de la página. Aprovecharlas de manera efectiva es clave para una estrategia SEO sólida y un sitio web accesible.

    Relación de Meta Etiquetas y SEO

    La relación se basa en que las meta etiquetas son elementos relevantes para la estrategia de SEO de cualquier empresa, ayudan en el posicionamiento y, en consecuencia, en el flujo de visitantes para el sitio web.
